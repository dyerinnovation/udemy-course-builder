# Udemy Course Creation Guide

## Course Planning
1. Define target audience (beginner/intermediate/advanced)
2. Define learning objectives (3-5 clear outcomes)
3. Research competition (existing Udemy courses on the topic)
4. Plan course length (certification courses: 8-15 hours; skill courses: 3-8 hours)
5. Create outline before recording anything

## Course Structure Best Practices
- Sections: 8-15 sections for a full course
- Lectures per section: 3-8 lectures
- Lecture length: 5-15 minutes (sweet spot is 7-10 minutes)
- Include a quiz after every 2-3 sections
- Start with a "Welcome and Course Overview" section
- End with a "Next Steps and Resources" section
- Include at least one hands-on project or lab section

## Recording Standards
- Audio: Clear, no background noise, consistent volume
- Video: 1080p minimum, 16:9 aspect ratio
- Screen recording: Clean desktop, large font size, dark theme preferred
- Webcam: Optional but recommended for intro/outro
- Editing: Remove long pauses, filler words, and mistakes

## Content Quality
- Every lecture should teach ONE concept
- Show, don't tell — use demos, code, diagrams
- Provide downloadable resources (cheat sheets, code, slides)
- Include real-world examples and scenarios
- Practice questions after key concepts

## Launch Checklist
- [ ] All lectures recorded and uploaded
- [ ] Course description written (keyword-optimized)
- [ ] Promotional video created (2 minutes max)
- [ ] Course image designed (750x422px)
- [ ] Pricing set
- [ ] Landing page reviewed
- [ ] At least one practice test (for cert courses)
- [ ] All resources uploaded
- [ ] Course submitted for Udemy review
