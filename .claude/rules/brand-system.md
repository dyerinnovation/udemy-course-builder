# Dyer Innovation Brand System

All generated content must follow the Dyer Innovation brand guidelines:

- **Primary color:** Leaf Green `#3CAF50`
- **Secondary:** Deep Green `#1B8A5A`, Teal `#0D7377`
- **Text:** Dark Navy `#1A3A4A` (headings), Near Black `#111928` (body)
- **Backgrounds:** Light Mint `#E8F5EB` (content), White (body areas)
- **Accent:** Amber `#E3A008` (exam tips, warnings)

Typography uses system fonts: Calibri for headings, Segoe UI / Helvetica for body.
